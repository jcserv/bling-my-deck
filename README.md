# bling-my-deck

![visitors](https://img.shields.io/endpoint?url=https://vu-mi.com/api/v1/views?id=jcserv/bling-my-deck)

bling-my-deck is a simple web application to help Magic: The Gathering users find the maximum ceiling
for their pet decks.
